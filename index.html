KathoolXYZ Main Server Shutdown

This has been written in a short amount of time to break the terrible news to all KathoolXYZ private access members and public users.
Unfortunately, in a time of major development and progress with our services such as the Cloud service and Minecraft server that we have hosted using our main server, the "Clover" server. It has come to a point where we have been forced to shut the "Clover" server down. Does this mean we will not return? No, but as of right now our only server available has been required to be decommissioned. This is not only an announcement for all users who may be inconvenienced by this event but also a dedication to this project. I, the owner of KathoolXYZ, have had a wonderful time working on this and sharing the return with not only my friends but the public. ALL of our services will return in the future but I must redirect my focuses onto school, I am one person, I can only do so much and spend so much on these FREE services. Thank you for your understanding.
RIP "Clover" Server 2023 - 2023
